What is physical computing?

Computing that senses a wide array of interaction from the physical world.

Interaction = Input (e.g. listening, seeing, feeling) + Process (thinking) + Output (talking, moving)

How do we encounter physical computing?

  • Sensors for light, movement, sound, temperature.
  • Inputs such as switches, sliders, knobs, buttons
  • Wearable electronics

Why Might I want to learn physical computing?

  • Have a more nuanced understanding and appreciation of the presence of physical computing around you
  • Prototyping ideas: toys, art, tools
  • Creating/recreating historical artifacts

Physical Computing and the Humanities

  • Early wearables kit
  • The Wonderboy

Getting Started with Arduino

Getting Started with Arduino

  • Find your Arduino board, breadboard, and USB cable in your kit.
  • Connect the Arduino board to your computer with the USB Cable.
  • The Arduino should get power, and have a few lights on, if working.

Arduino Integrated Developer Environment (IDE)

Download the IDE for your platform and install.

Open the Arduino IDE

Setting up the Arduino IDE

  • Navigation to Tools → Board and select "Arduino Uno" from the list.
  • Navigate to Tools → Serial Port
    • Mac: Select "/dev/ttyusbmodem"
    • Windows: Select "com3" or higher.

Download Sparkfun Inventor's Kit Code

Download a Zip or check out the Github repo.

Circuit #2: Potentiometer

A potentiometer is a variable resistor, that can change the amount of voltage used.

Examples: Volume controls, rotary light dimmers.

Circuit #2: Analog vs. Digital

  • Digital: Great for reading "On" and "Off". Good for input and output.
  • Analog: Great for reading/setting a range of states; Good only for inputs, to change output
