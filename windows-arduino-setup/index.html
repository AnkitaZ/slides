Install the Arduino driver on Windows 7

Step 1

Is the Arduino recognized?

When you plug in the USB for the Arduino the first time, the system will try to locate the correct drivers.

The system will not be able to find them, so we'll have to install them manually.

Step 2

Open the Control Panel

We need to access the "unrecognized" hardware in order to install the driver.

To do this open the Control Panel.

Step 3

Click on System settings

Step 4

Click the Device Manager

Step 5

Right click the unkown device, then select the Update Driver Software

Step 6

Select Browse my Computer

Step 7

Click the Browse button

Step 8

Select the drivers folder

Go to the C: drive, then to the arduino-1.6.5-r2 (or whatever version of the arduino software is installed) folder, and then to the drivers folder.

Click on the drivers folder and make sure that drivers shows up in the Folder: window.

Click the OK button.

Step 9

Select Next

The path to the drivers folder should now be in the location box.

Now click the next button.

Step 10

Click the install button

Step 11

Wait for the driver to be installed

Step 12

Close open windows

Once the drivers are installed, you can safely close the install driver window, and any other windows that were opened.

Step 13

Open the Arduino program

Step 14

Select the correct port

With the driver installed, we can select the correct port. This is usually com3

Step 15

Select the correct Arduino board

Finally, select the correct Arduino board that you are using. Most likely this is the Arduino Uno
